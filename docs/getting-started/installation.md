## Installation

### Using pip
To install the Axon SDK, simply run:

```bash
pip install axon-sdk
```

Alternatively, install `axon-sdk` in editable mode (ideal for development):

```bash
git clone https://github.com/neucom/axon-sdk.git
cd axon-sdk
pip install -e .
```
